# Mira Studio App

Projeto acadêmico desenvolvido para a disciplina de Programas para Dispositivos Móveis.

## Objetivo

Desenvolver um protótipo de aplicativo móvel inspirado nos processos observados no salão de beleza Mira Studio, aplicando conceitos de desenvolvimento Android, Kotlin e Firebase.

## Funcionalidades

- Cadastro de clientes
- Agendamento de serviços
- Integração com Firebase
- Interface para dispositivos móveis

## Tecnologias Utilizadas

- Kotlin
- Android
- Firebase Firestore
- VS Code
